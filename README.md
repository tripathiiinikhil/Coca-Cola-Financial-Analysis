# Coca-Cola Financial Analysis (2009–2018)

## Project Overview

This Excel project analyzes Coca-Cola's financial performance from FY2009 to FY2018 using Revenue, Net Income, and Profit Margin metrics.

## Objectives

* Analyze revenue trends over time
* Evaluate profitability using Net Income
* Calculate and compare Profit Margins
* Identify key business insights and performance trends

## Tools Used

* Microsoft Excel
* Formulas and Functions
* Data Visualization
* Financial Analysis

## KPIs

* Highest Revenue: FY2012
* Highest Net Income: FY2010
* Lowest Profit Margin: FY2017
* Recovery Year: FY2018

## Key Insights

1. Revenue reached its highest level in FY2012.
2. Net Income peaked in FY2010.
3. Higher revenue did not always result in higher profitability.
4. FY2017 recorded the lowest profit margin.
5. Profit margin recovered in FY2018.
